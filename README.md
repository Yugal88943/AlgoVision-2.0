# 🚀 AlgoVision 2.0  
### 📊 A DSA-Based Smart Analytics Platform

---

## 📌 Overview

**AlgoVision 2.0** is a unified analytics platform built using core **Data Structures and Algorithms (DSA)** to simulate a real-world smart food ordering and analytics system.

The project demonstrates how multiple DSA concepts can be integrated into a single system to solve real-world problems like order processing, searching, filtering, and optimization.

---

## 🎯 Key Objectives

- Integrate multiple DSA concepts into one system  
- Simulate real-world workflows (like food delivery platforms)  
- Provide efficient data handling and optimization  
- Build an interactive dashboard using Java Swing  

---

## 🧠 DSA Concepts Used

| Concept | Usage |
|--------|------|
| 📥 Queue | Order processing (FIFO) |
| ⚡ HashMap | Fast data storage and retrieval |
| 📦 Array | Catalog management |
| 🔍 Searching | Item lookup (linear search) |
| 🎯 Greedy Algorithm | Cheapest item selection |

---

## 🏗️ System Architecture

User (UI - Dashboard)  
        ↓  
Order Input → Queue (FIFO Processing)  
        ↓  
HashMap (Storage: Users & Orders)  
        ↓  
Array (Food Catalog)  
        ↓  
Greedy Algorithm (Optimization)  
        ↓  
Analytics Output (Dashboard)

---

- 🔎 Search response time: < 2 ms

## 🚀 Getting Started

### 📥 Clone the Repository

```bash
git clone https://github.com/Yugal88943/AlgoVision-2.0
```

---

### 📂 Navigate to Project Folder

```bash
cd AlgoVision-2.0
```

---

### ⚙️ Compile the Project

```bash
javac ui/Dashboard.java
```

---

### ▶️ Run the Application

```bash
java ui.Dashboard
```
